Setup flow

Use this page to find which wheel axes and buttons should control the FPV RC Viewer.

  1. Connect the wheel or controller to this device, then click this page once.
  2. Move only one control at a time. The row that lights up is the axis or button to assign.
  3. Set Steering, Throttle, Brake, Drive, and paddle buttons from the buttons under each row.
  4. Use invert, gain, and deadzone if the direction or sensitivity is wrong.
  5. Press Save for Viewer, then open the Viewer. The saved mapping is used automatically.
